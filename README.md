> ### 大二下做的博客



**自己写的博客 技术栈** ： 

- 页面：h5+css3+js+jquery 
- 渲染模板：art-template
- 后台: node.js express mongoose（连接，操纵mongodb ）
- 数据库：mongodb 
- 服务器：阿里云上买的（上线时推荐用宝塔）
